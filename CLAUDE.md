# Project: AI Design Practice

## Who I am
Product designer at TELUS Digital. Learning AI assisted workflows. Not a developer, explain code decisions briefly when you make them.

## Stack
HTML, CSS only. No frameworks, no libraries unless I ask.

## Rules
- Always work on a branch, never commit to main
- Keep code clean and readable
- When creating UI, default to dark background #1a1a1a, off-white/creamy text, Clash Display for Display and headings (h1-h3) and Epilogue
- File names use kebab-case: my-file-name.html

## What this project is for
A practice/exploration repo for learning git + Claude Code workflows.
